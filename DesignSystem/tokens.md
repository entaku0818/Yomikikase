# デザイントークン

すべての色・余白・角丸・影・タイポの正典。**ここにない値は使わない。** 追加が必要なときは、まずこのファイルを更新する。
実装上の出どころは `AppTheme.swift`（色）と各 View のローカル定数（余白・角丸）。

## カラー

| トークン | ライト | ダーク | 用途 | コード |
|---|---|---|---|---|
| `primary` | `#4B47E0` | `#8B87FF` | 再生ボタン・全アイコン・CTA・タブのアクティブ・進捗バー・選択中フィルタ | `Color("AccentIndigo")` |
| `primarySoft` | primary 12% | primary 12% | アイコン地・選択地 | `primary.opacity(0.12)` |
| `onPrimary` | `#FFFFFF` | `#FFFFFF` | アクセント上の文字・アイコン | `Color.white` |
| `groupedBg` | `#F2F2F7` | `#000000` | グループ背景 | `Color(.systemGroupedBackground)` |
| `card` | `#FFFFFF` | `#1C1C1E` | カード / セル | `Color(.secondarySystemGroupedBackground)` |
| 本文 | `#1C1C1E` | `#FFFFFF` | 本文テキスト | `Color.primary` |
| 補助 | label 60% | label 60% | 補助テキスト | `Color.secondary` |
| 区切り線 | separator | separator | セパレータ | `Color(.separator)` |

**禁止**: `Color.indigo` / `Color.orange` / `.blue` / `.red` などの直書き。グラデーション。多重影。

## 角丸

| 対象 | 半径 |
|---|---|
| カード / シート | 16 |
| セル / タイル | 14〜18 |
| アイコン地 | 11〜15 |
| ピル | 100 |

## 余白

| 対象 | 値 |
|---|---|
| 画面端（基本） | 左右 20 |
| リスト内テキスト | 16 |

## 影

- カード: `0 1 3 / 5%`（極薄 1 段のみ）
- ダーク: 影を消し **0.5px の区切り線**で代替
- 多重影・濃い影は禁止

## タイポグラフィ

| 役割 | サイズ / ウェイト |
|---|---|
| 大タイトル | 32 / 700 |
| 見出し | 20 / 700 |
| 本文 | 16 / 400 |
| 補助 | 13〜14 / secondary |
| 最小フォント | 12 |

## サイズ規約

| 対象 | サイズ |
|---|---|
| ホームタイルのアイコン地 | 52×52 / 角丸 15 |
| リスト行のアイコン地 | 40×40 / 角丸 11 |
| 再生ボタン（PlayerBar） | 直径 66pt・アクセント塗り・白アイコン |
| タップ可能要素の最小ヒット領域 | 44×44pt |
