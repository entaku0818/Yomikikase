# Voice Narrator (VoiceYourText)

![Version](https://img.shields.io/badge/version-0.17.0-blue)
![Platform](https://img.shields.io/badge/platform-iOS-lightgrey)
![Swift](https://img.shields.io/badge/Swift-5.9-orange)

テキスト・PDF・EPUB・Webページなど多様なコンテンツを音声で読み上げるiOSアプリです。

## 機能

| コンテンツ | 説明 |
|---|---|
| テキスト入力 | 直接テキストを入力して読み上げ |
| PDF | PDFファイルのテキスト抽出・読み上げ |
| TXTファイル | ローカルテキストファイルのインポート |
| Google Drive | Google DriveからファイルをインポートしてTTS |
| EPUB（本） | EPUBファイルのインポートと読み上げ |
| スキャン | カメラスキャンによるOCRテキスト読み上げ |
| Webリンク | URLからWebページのテキストを抽出して読み上げ |

- 多言語対応（10言語）
- 読み上げ速度・音声の高さ調整
- プレミアムサブスクリプション（広告非表示・無制限ファイル数）
